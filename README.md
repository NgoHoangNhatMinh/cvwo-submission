# CVWO 2025 Assignment - Forum

## Local Installation

1. You need the following prerequisites:
   - [Ruby 3.3.0](https://www.ruby-lang.org/en/)
   - [Rails 8.0.1](https://rubyonrails.org/)
   - [Bundler](https://bundler.io/)

2. Clone the repo:
   ```console
   $ git clone https://github.com/NgoHoangNhatMinh/cvwo-submission.git
   ```

3. Installing dependencies:
   ```console
   $ bundle install
   ```

4. Start the server:
   ```console
   $ rails s
   ```
   ```console
   $ npm run dev
   ```

5. Open the web application from the link `http://localhost:5173`
